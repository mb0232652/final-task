Git

  • git init- создаёт новый Git репозиторий в текущей папке
  • git add- добавить файлы в очередь на сохранение
  • git commit -m "сообщение"- сохраняет изменения с комментариями
  • git push- отправляет изменения на GitHub
  • git stftus- показывает состояние файлов в репозитории
  • git log- показывает историю комитов
